# Smart Control Energy Management System (Classroom Automation)
## Overview
This project demonstrates a Smart Energy Management System for a classroom scenario.  
The system automatically controls lights and fans based on student entry and exit using IR sensors and Arduino Nano.
When a student enters → Lights & Fan turn ON  
When a student exits → Lights & Fan turn OFF  
This helps in reducing unnecessary power consumption and improves energy efficiency.
---
## Components Used
- Arduino Nano
- IR Sensors (Entry & Exit)
- 9V Battery
- Buck Converter
- Switch
- Small DC Fan
- LEDs (for Bulbs demonstration)
- Resistors
- Jumper Wires
---
## Working Principle
- Two IR sensors are placed at the entry and exit gates.
- Entry sensor detects incoming students.
- Exit sensor detects outgoing students.
---
## Power System
- 9V battery used as primary source.
- Buck converter steps down voltage for Arduino and components.
---
## Programming
- Arduino Nano programmed using Arduino IDE.
- Logic includes:
  - IR sensor detection
  - Student counter increment/decrement
  - Output control for fan and LEDs
